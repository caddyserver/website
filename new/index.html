Experience it

Discover Caddy's automagic HTTPS features in this demo!

Point any subdomain named caddydemo to:

A 123.123.123.123 AAAA ::f00

Then visit it in your browser.

You'll notice how Caddy provisions a certificate for your domain automatically.

Not working? Make sure to use a subdomain of a registered domain, not a "sub-subdomain." Verify you have created the public DNS records shown above with the correct values. You may have to wait to allow time for propagation. Consult your DNS provider's documentation or support for more information.

Example subdomains that could work: caddydemo.example.net, caddydemo.example.co.uk

(Caddy is capable of serving TLS for any domains and IPs. This demo is arbitrarily restricted.)
